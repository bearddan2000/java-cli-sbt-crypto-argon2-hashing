# java-cli-sbt-crypto-argon2-hashing

## Description
A demo for Argon2 one-way hashing of
a password.

## Tech stack
- java
- sbt

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
